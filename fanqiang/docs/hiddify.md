#### 什么是WARP

**WARP** 全称为 Cloudflare WARP 是 Cloudflare 推出的一项免费 VPN 服务。 WARP 旨在提供更安全、更快速的互联网连接，保护用户的在线隐私和安全。由于 Cloudflare 有着全球最强大的 IP 库，所以基本不用担心被封锁的问题。
